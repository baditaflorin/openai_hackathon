Deployment

Docker and host runtime guidance

Use the same settings model across local development, Docker demos, and hosted deployments.

Compose with Ollama

Start the integrated local AI stack with the optional local-ai profile.

docker compose --profile local-ai up --build

Build with local Whisper

Install the optional Whisper dependency directly into the web image for CPU or Linux GPU deployments.

docker build --build-arg INSTALL_LOCAL_WHISPER=true -t clipmato:0.4.0 .

Apple Silicon

Run Clipmato on the host to use the Apple mps device. Docker Desktop containers do not expose the Apple GPU runtime.

CLIPMATO_TRANSCRIPTION_BACKEND=local-whisper clipmato-web
